## How to Contribute

1. Fork the repository
2. Create a new branch
3. Make your changes
4. Test thoroughly
5. Submit a pull request

## Code Style

- Follow existing code style
- Use meaningful variable names
- Add comments for complex logic
- Test your changes locally

## Adding Malayalam Trolls

Edit `api/index.py` and add to the `trolls` dictionary:

"SectionName": [
"Your funny Malayalam troll here!",
]
